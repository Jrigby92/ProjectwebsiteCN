Plan and carry out tasks using IT - how do you select the right kind
of technology for the job.











First I think it’s important to look at the end goal, which in this case is to create a showcase website portfolio. After this is established I can begin to evaluate the technological requirements of reaching this goal.

The initial most outstanding requirement was to find an effective code editor. I already had one in mind, but if I didn’t it would be easy to discover via search engines or word of mouth, that visual studio code editor is the number one go to code editor for beginners and the majority of professionals. The next steps would be to decide on a coding language, or combination of coding languages to complete the job. Unsurprisingly, there is a wealth of free information and tutorials online which relate to web development and website building. However, choosing information thats sound and relevant to my task is more difficult. On this occasion, because I am a beginner at website building I opted for the most popular simple option. Which was to build a website using a combination of HTML and CSS, although I had a willingness to delve into javascript if necessary. I knew from my research that the use of HTML and CSS code languages would be enable me to create a well formatted and beautiful website, if used properly. I felt confident about this choice due to the large amount of online support for beginners in these two languages.


Additionally, through my studying and research I have discovered the existence of libraries such as ‘bootstrap’. I am part way through the development of my website already and have yet use this resources, yet either bootstrap or another css library will likely constitute my next choice of technology for my project.

Finally, to cover the obvious choice of computer hardware for this project. My decision was based on what I already owned, which is a Mac computer. For future projects that might have larger budgets, I would focus much more time on selecting the correct hardware for the job. Again by assessing my requirements, establishing a budget and setting these two factors as my parameters for choosing hardware. I would then conduct research using methods outlined in question 1.5, to find the best available software within my established parameters.
