Review the ongoing use of IT tools and change approach as needed – in making
your own website, are you happy with the tools you used?















Thus far the tools I’ve used such as: Visual Studio code, Trello, Photoshop, Google Chrome browser etc have been sufficient for me to create my website.

Although I am still working on the basic front-end of the website, later I plan to introduce deeper functionality and at that point I may need to expand my tool kit. However, up to this point the tools I’ve used have been sufficient. Visual studio code for example provides many helpful functions such as, the programs ability to automatically spot syntax errors, an easy workflow between files, in program linking between html and css sheets and early predictions as to which properties or functions etc your going to type next. These features have allowed me to write, adapt and replicate large sections of code very quickly. The program is also free to use which is one of the many reasons I plan to keep it in my toolkit. Additionally, through my studying and research I have discovered the existence of libraries such as ‘bootstrap’. I am part way through the development of my website already and have

yet to use this resources, but it's likely that either 'Bootsrap or another CSS library will constitute my next choice of technology for my project.

Trello has also been helpful in planning and keeping up to date with my project, whilst maintaining a flexible point of contact to share work with my tutors. It too has helpful features, such as the ability to produce lists with easily transferable items. Also the ability to label each item with custom labels, and share the trello board link with others have been helpful. I have been using notepad as text editor and google chrome as a way to check in the appearance and functionality of my website in its final form. All of this programs have been fit for purpose and have allowed me to move on with my project in good time. I will re-review this conclusion when I add more functionality to the website.
