2.3 Assess the strengths and weaknesses of your final project – are you happy with your website?



Thus far I am happy with the tools I have implemented on the website, however I do plan to add more functionality before completion.



Up to this point I feel the website is user friendly, well presented and functional enough to act as a portfolio / showcase of my knowledge. In terms of website navigation I have so far implanted a functional ’sticky’ menu bar tool, along with a functional name/logo in the top left of every page which diverts users back to the about me page. For presentation, I have implanted responsive videos, images and graphics to several of the pages. In addition to well formatted text to display my question answers. I have also customised the background colours of each page.


However, my next goal for the website presentation will be to add more complex backgrounds using ‘flex’. I hope this will add a new dynamic and deepen the already broad aesthetics of the website. I also plan to make the website tablet and phone repsonsive. In regards to enhancing the functionality of the website, I plan to add a functional contact page for clients / employers to contact me through. Although I anticipate that this will require some java script coding and might be time intensive, hopefully I will have enough time to add a functional contact page with an interactive map.
