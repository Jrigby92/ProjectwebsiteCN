I hope to maintain and build upon the already established standard, for the remainder of the website.

In detail

At this point I have only created webpages for half of the written questions. After having already defined a broad and cohesive style in this first half of the website, I hope to improve upon the website as a whole by maintaning the same standard for the remaining webpages. I hope to further improve this standard with additional functionalities such as a contact page with an interactive map.

If the time was available, I would like to experiment with ‘flex’ classes and boxes. I believe this CSS functionality holds huge potential for the design and aesthetics of website creation. I would also like to experiment further with Javascript by adding more complex functions to the website. Perhaps information forms to collect data on visitors or introducing short games on parts of the website to keep users entertained. Interactive design elements would be another interesting addition, perhaps by allowing the background color to change every time the user clicks on the webpage.
